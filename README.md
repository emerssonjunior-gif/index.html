# Agrinho 2026

## Descrição
Este projeto foi desenvolvido para o **Agrinho 2026** com o objetivo de valorizar o agronegócio da região. Ele destaca a importância do café e do trabalho rural, mostrando a dedicação dos trabalhadores no campo.

O projeto foi criado por **Emersson, do 3º B, do Colégio Leonardo**.

---

## Funcionalidades
- Página web com **HTML, CSS e JavaScript** integrados.
- Seções sobre o **agronegócio da região** e a **importância do café**.
- Galeria de imagens com **pé de café** e **trabalhadores rurais**.
- Botão interativo que exibe uma **mensagem de valorização do campo**.
